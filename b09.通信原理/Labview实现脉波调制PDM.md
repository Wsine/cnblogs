# Labview实现脉波调制( PDM )

根据定义为脉冲宽度调制
生成一个正弦信号，得到其幅值输入给一个方波信号的占空比
由于方波信号的占空比里面含有正弦信号的信息
因此通过滤出方波信号的占空比信息则可以恢复波形
